# AI Newsletters Skill

A Claude Code skill for fetching and summarizing daily AI news, research, and industry developments.

## Installation

1. Ensure this skill is in your `~/.claude/skills/` directory
2. The skill will be automatically available in Claude Code

## Usage

Simply ask Claude for AI news using natural language:

- "What's new in AI today?"
- "Any new AI models this week?"
- "Latest AI research papers"
- "Show me today's AI news"
- "OpenAI announcements"
- "AI regulation updates"

## Features

- 🔍 Smart search across authoritative AI sources
- 📰 Categorized news (Models, Research, Tools, Industry, Policy)
- 🔗 Source attribution with direct links
- 📅 Flexible time ranges (today, this week, this month)
- 🎯 Topic-specific filtering
- ✅ Quality-focused curation

## Output Format

Structured markdown summaries with:
- Major releases and models
- Research papers and breakthroughs
- Tools and frameworks
- Industry news and funding
- Policy and regulation updates
- Key insights and trends

## Sources

Prioritizes authoritative sources including:
- arXiv.org for research
- Official company blogs (OpenAI, Anthropic, Google, Meta)
- Reputable tech outlets (The Verge, MIT Tech Review)
- Academic journals and conferences
